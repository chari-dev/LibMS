A Simple Library Management System (using LAMP Stack)

Setup:
> CD Desktop
> 
> MKDIR LibMS
> 
> CD Libms
> 
> git clone https://github.com/chari-dev/LibMS.git
> 
> docker-compose up --build
> 
> open localhost:9001 (PHPmyAdmin, mysql_db, root, root)
> 
> open localhost:9000/index.php
